# Cloned Repositories

## Multilingual Latent Concepts
- **URL**: https://github.com/qcri/multilingual-latent-concepts
- **Location**: `code/multilingual-latent-concepts/`
- **Purpose**: Provides tools for activation extraction, clustering, and concept alignment across languages.
- **Key Files**: 
    - `get_alignment.sh`: Main script for alignment.
    - `code/alignClusters.py`: Python logic for cluster alignment.
    - `NeuroX/`: Submodule for neural representation analysis.

## False Friends
- **URL**: https://github.com/dhfbk/falsefriends
- **Location**: `code/falsefriends/`
- **Purpose**: Contains datasets for false friends and cognates in Italian, English, and French.
- **Key Files**:
    - `data/test-en.txt`: Annotated IT-EN word pairs.
    - `data/test-fr.txt`: Annotated IT-FR word pairs.
