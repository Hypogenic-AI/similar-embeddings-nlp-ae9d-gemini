\subsection{Implementation Details}
We use the `sentence-transformers` library to load the `paraphrase-multilingual-MiniLM-L12-v2` model. Embeddings are extracted from the `[CLS]` token for single words. The code is available at \url{https://github.com/hypogenic/similar-embeddings-nlp}.
